# Car Rentals Project

## Backend (ASP.NET Core)
- Navigate to `CarRentals_JEY`
- Run `dotnet restore` and `dotnet run`
- Runs on `https://localhost:7127`

## Frontend (Angular 16)
- Navigate to `CarRentals-UI`
- Run `npm install` and `ng serve`
- Runs on `http://localhost:4200`

The frontend consumes the API endpoints for car listing and reservations.
